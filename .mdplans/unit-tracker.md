## Data Entry Notes

### Source
- Battle Nations Wiki: https://battlenations.miraheze.org/wiki/Units

### Per-Unit Data Required
1. Basic info (id, name, description, imageUrl, category, type, unlockLevel, productionTime, motto)
2. Production cost (Resource[])
3. Stats per rank (UnitRank[])
4. Promotion costs per rank (PromotionCost)
5. All actions with rank-based damage/stats (UnitAction[])
6. Special abilities (string[])

### Workflow
1. Open unit's wiki page
2. Extract all data fields
3. Format as JSON matching gameTypes.ts
4. Add to units.json
5. Check box in this tracker
